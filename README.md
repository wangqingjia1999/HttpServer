High performance HTTP/1.1 server on Linux.

## Features
* HTTP Methods
  - [x] POST
  - [x] GET
  - [ ] PUT
  - [ ] PATCH
  - [ ] DELETE

* Benchmark
* I/O Multiplexing

## Build Environment
* Linux Operating System
* GCC 9.3.0
* [Cmake](https://cmake.org/)
* [Googletest](https://github.com/google/googletest)

## How to build? 
```shell
git clone https://github.com/Bitate/HttpServer
cd HttpServer
bash build.sh
```
